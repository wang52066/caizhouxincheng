# 🚀 蔡州芯辰官网 · GitHub Pages 部署指南(3 分钟搞定)

你之前打不开的链接是临时的沙箱链接,会过期。
**跟着这个指南做,3 分钟后你就有了一个永久的、不会过期、不会被拦截的网站链接。**

---

## 📋 准备工作(1 分钟)

打开浏览器,访问: **https://github.com**

- 如果你已有 GitHub 账号 → 直接登录
- 如果没有 → 点「Sign up」注册(用邮箱,免费)

---

## 🎯 部署步骤(2 分钟)

### 第 1 步:创建新仓库

1. 登录后,点右上角 **「+」→「New repository」**
2. 填写:
   - **Repository name**(仓库名):`caizhouxincheng`(必须填这个,后面链接要用)
   - **Description**:选填,可不填
   - **Public**(必须选这个!)
   - **不要**勾选「Add a README file」
3. 点 **「Create repository」**

> ⚠️ 仓库名一定要是 `caizhouxincheng`,因为这样你的网址会是:
> `https://你的用户名.github.io/caizhouxincheng/`

---

### 第 2 步:上传文件

1. 进入新创建的仓库页面后:

2. 找到 **「uploading an existing file」** 链接(在中间位置,蓝色小字),点它
   > 或者点 **「Add file」→「Upload files」**

3. 把 caizhouxincheng-website 文件夹里的内容全部拖进浏览器上传窗口

4. 滚动到底部,找到 **「Commit changes」** 按钮,直接点

---

### 第 3 步:开启 Pages 服务

1. 在仓库页面,点顶部 **「Settings」**
2. 左侧菜单找到 **「Pages」**
3. 在「Branch」那里: 选择 main / /root -> Save
4. 等 1-2 分钟,页面会刷新,出现绿色提示: Your site is live at https://你的用户名.github.io/caizhouxincheng/
